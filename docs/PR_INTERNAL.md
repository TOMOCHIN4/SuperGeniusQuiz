# 超天才クイズ v4 - 社内広報資料

## 概要

「超天才クイズ」は、小学生の学習をサポートするWebベースのクイズアプリケーションです。国語・算数・理科・社会の4教科を網羅し、ゲーム感覚で楽しく学べる設計になっています。

## 開発背景

子どもたちが自発的に学習に取り組める環境を提供するため、以下のコンセプトで開発しました：

- **楽しさ優先**: ゲーミフィケーション要素を取り入れ、学習を遊びに変える
- **即時フィードバック**: 回答直後に正誤がわかり、達成感を得られる
- **継続性**: 進捗の可視化により、モチベーションを維持

## 主な機能

### 1. クイズ機能
- **4教科対応**: 国語、算数、理科、社会
- **ジャンル別出題**: 各教科内で細かいジャンルを選択可能
- **タイマー制**: 制限時間内に回答するスリル感
- **即時フィードバック**: 正解/不正解を視覚的に表示

### 2. 学習管理
- **個人統計**: 正答率、解答数をリアルタイム表示
- **履歴記録**: すべての回答履歴を保存
- **弱点把握**: 教科別の正答率で苦手分野を特定

### 3. コンテンツ管理
- **大規模問題データベース**: 3,095問を収録
- **新着問題通知**: 追加された問題をすぐに挑戦可能
- **使用頻度ベース出題**: まだ解いていない問題を優先的に出題

## 技術構成

| 項目 | 技術 |
|------|------|
| フロントエンド | HTML/CSS/JavaScript |
| バックエンド | Google Apps Script (GAS) |
| データベース | Google Sheets |
| ホスティング | Google Apps Script Web App |
| 問題生成 | Gemini API (gemini-3-flash-preview) |

### アーキテクチャの特徴

- **サーバーレス**: Google インフラを活用し、運用コストゼロ
- **スケーラブル**: Google Sheetsの容量まで拡張可能
- **メンテナンス容易**: スプレッドシートで問題データを直接編集可能

### ジャンル構成

**国語**: 漢字・語彙、文法、物語文読解、説明文読解、詩・短歌・俳句、作文・表現、ことわざ・慣用句、知識・文学史

**算数**: 計算、数の性質、割合・比、速さ、単位・量、平面図形、立体図形、場合の数・確率、規則性、文章題

**理科**: 力・運動、電気、磁石、物質の性質、水溶液、燃焼・気体、植物、動物、人体、天体、気象、環境
